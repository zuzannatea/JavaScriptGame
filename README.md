# JavaScript Game

### A browser-based game built with JavaScript & Flask for **CS1116**

This project is a full-stack browser game developed as coursework for the **CS1116 module** at **UCC**.

---

## Main Features

- Dijkstra-based pathfinding system
- Procedural level generation
- Combat system based on player decisions
- Multiple enemy types implemented using object-oriented inheritance
- Level progression with increasing difficulty

---

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (ES6)

### Backend
- Python
- Flask

---

## Installation & Running

### 1. Clone the repository

```bash
git clone https://github.com/zuzannatea/JavaScriptGame.git
cd JavaScriptGame
```

### 2. Install dependencies
Ensure Python is installed, then install Flask:
```bash
pip install flask
```

### 3. Run the backend server
```bash
flask run
```
